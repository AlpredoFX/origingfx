# OriginGFX

> Digital Gallery for Minecraft Artists

OriginGFX adalah galeri digital yang menampilkan karya-karya dari para kreator Minecraft. Berawal dari portfolio pribadi, kini menjadi ruang bagi para artist untuk memperkenalkan diri dan karyanya.

## Filosofi

**Gallery First. Store Second. Artist Always.**

Kami percaya bahwa sebuah karya layak ditemukan sebelum dijual. Artist bukan sekadar penjual, mereka adalah kreator, pencerita, dan pembangun dunia melalui karya mereka.

## Tech Stack

- HTML5
- CSS3 (Custom)
- JavaScript (Vanilla)
- Laravel (Future)
